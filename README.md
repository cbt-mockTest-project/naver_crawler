## 네이버 블로그 노출순위 봇

네이버 블로그 노출순위를 알려주는 서비스입니다.

### STACK

- Client : Nextjs, Tailwind, PWA, Graphql, Apollo
- Server : Nestjs
- Deploy : Vercel, Aws

### DESCRIPTION

블로그 이름과 검색어를 입력하면, 네이버와 다음에서의 블로그 노출 순위를 알려줍니다.<br/><br/>
