## 네이버 블로그 노출순위 봇

네이버 블로그 노출순위를 알려주는 서비스입니다.

### STACK

- Client : Nextjs, Tailwind, PWA
- Server : Nestjs
- Deploy : Vercel, Aws-lightsail

### DESCRIPTION
블로그 이름과 검색어를 입력하면 , 전체탭과 블로그탭에서의 노출 순위를 보여줍니다.<br/><br/>
